### 重要技術筆記：Google News 連結
- 從 RSS `<link>` tag 抓取的完整連結（含 ?oc=5）可正常訪問（200）
- 截斷版連結（只有 70 字元）會 400
- 取連結方式：`re.search(r'<link>([^<]+)</link>', item)` 不能截斷
- 驗證方式：用 urllib 測試每條連結是否 200 才推上 GitHub
